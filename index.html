Features

Built for your library and your privacy

Organize books from your own folders, keep everything on your machine, and read without handing data to anyone else.

Library management

Add books your way

Import one file at a time, pull in a whole folder, include nested subfolders, or set up folder sync so your library stays up to date automatically.

  • Add File — single EPUB or PDF
  • Add Folder — bulk import
  • Include Subfolders
  • Sync Folders — auto-scanned directories

Library management

Edit book metadata

Fix titles, authors, and cover art after import so large collections stay organized. Premium lets you curate every detail of your shelf.

  • Edit title and author
  • Replace cover images
  • Reset to defaults when needed

Your library

Browse and organize at a glance

See covers, titles, authors, and formats in a sortable list. Mark favorites, switch views, and manage growing collections without losing track of a single book.

  • Sort and filter your collection
  • List and grid-style browsing
  • Light and dark interface themes

Reading experience

Typography and layout controls

Open the styling panel while you read to adjust fonts, spacing, and margins. Dial in a comfortable layout for long sessions on any screen.

  • Font family and size
  • Line, word, and letter spacing
  • Margin presets and page layout
