# Obsidian Ebook Reader

A lightweight, privacy-focused EPUB reader designed for a distraction-free reading experience.

[View the Live Site](https://www.obsidianreader.app)

## Features
* **Privacy First:** No tracking, no cloud syncing. Your books stay on your device.
* **Format Support:** Optimized for EPUB 2 and EPUB 3 standards.
* **Customization:** Adjustable fonts, margins, and full Dark Mode support.
* **High Performance:** Built for speed with a minimal memory footprint.

## How to Use
1. Visit [yourdomain.com](https://www.obsidianreader.app) or the Microsoft Store to download the app.
2. Upload your EPUB files.
3. Start reading instantly.
